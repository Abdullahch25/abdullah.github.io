Previous Projects

Important person detection from multiple videos (Master's Thesis)

The goal of this work is to develop a unified framework for unsupervised person re-identification with application to important person detection from multiple videos. It focuses on topics such as
  • Face detection
  • Particle filters based object tracking
  • Face/clothing recognition
  • Detection of high level attributes (i.e. Race, Gender, Eyewears, Headwears, Hair colors, Age, etc.). An example can be seen here.
  • Sensor based temporal segmentation of videos
  • Supervised classification.
  • Hierarchical Agglomerative Clustering (HAC) with constraints (i.e. cannot-link constraints)
Tools used in this work: C++, OpenCV, Python, Intel's Threading Building Blocks (TBB), GNU Scientific Library (GSL)
